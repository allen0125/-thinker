# DropKindle

DropKindle 将 Dropbox 、坚果云、OneDrive中的文档推送到 Kindle

须将 drop2kindle#gmail.com ( #用at符号代替 ) 添加到 Kindle 白名单当中

依赖工具:

- Django
- Dropbox Python SDK
- SQLite
- Django-Crontab

支持文件类型:

- mobi
- azw
- txt
- doc
- docx
- pdf
ccccccgbbifrnnjblvvukfecjcikcvblgheutngfcvlt

文件大小限制: 20MB

推送频率: 5 分钟/次
